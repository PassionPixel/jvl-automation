# JVL Automation — Landing page

Site Next.js + Tailwind CSS prêt pour Vercel.

## Lancer en local
```bash
npm install
npm run dev
```

## Publier sur Vercel
1. Créer un compte sur vercel.com
2. Importer ce dossier via GitHub ou drag & drop
3. Ajouter le domaine jvlautomation.com dans Settings > Domains
4. Suivre les instructions DNS données par Vercel

## À modifier
- Remplacer `contact@jvlautomation.com` par ton adresse réelle si besoin
- Remplacer le bouton contact par ton lien Tally
- Ajouter tes cas clients/démos dès que tu en as
